Bekannte Probleme

Aktuell sind bei der Verwendung von Icon File Decorator folgende Probleme bekannt:
  • Die Aktivierung des Icon File Image Decorator führt dazu, dass für betroffene Dateiknoten die Overlay-Icons anderer Plugins (z.B. EGit) nicht mehr angezeigt werden.
