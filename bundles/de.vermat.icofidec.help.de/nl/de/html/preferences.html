Einstellungen

Hier kann man den Icon File Image Decorator und den Icon File Information Decorator seinen Anforderungen entsprechend konfigurieren.

Ein-/Ausschalten Icon File Image Decorator

Wenn man diesen Link anklickt, wird man auf die Einstellungsseite "General > Appearance > Label Decorations" weitergeführt, wo man den Icon File Image Decorator aktivieren oder deaktivieren kann.

Ein-/Ausschalten Icon File Information Decorator

Wenn man diesen Link anklickt, wird man auf die Einstellungsseite "General > Appearance > Label Decorations" weitergeführt, wo man den Icon File Information Decorator aktivieren oder deaktivieren kann.

Berücksichtigte Dateierweiterungen

Hier kann man eine Liste von Dateiendungen angeben, die die Dekoratoren berücksichtigen sollen.
Die Einträge werden ohne den führenden "." angegeben und durch Kommata getrennt. Es sind keine Wildcards zulässig. Die Namen werden nicht "case-sensitiv" behandelt.
Beispiel: "jpg,jpeg,png".

Achtung: Wird das Feld leer gelassen, werden alle Dateien betrachtet! Das kann sich negativ auf die Performance der Entwicklungsumgebung auswirken.

Berücksichtigte Image-Ordner

Hier kann man eine Liste von Namen von Ordnern angeben, in denen Dateien berücksichtigt werden. Dateien aus anderen Ordnern werden ignoriert (auch wenn ihre Erweiterungen passen).
Die Namen werden durch Kommata getrennt. Es sind keine Wildcards zulässig. Die Namen werden nicht "case-sensitiv" behandelt.
Beispiel: der Eintrag "icons,images" führt dazu, dass Dateien in allen Ordneren berüchsichtigt werden, deren Pfad einen der Strings "icons" oder "images" enthält.

Achtung: Wird das Feld leer gelassen, werden alle Dateien betrachtet! Das kann sich negativ auf die Performance der Entwicklungsumgebung auswirken.

Maximale Icon Breite

Die maximale Breite in Pixeln, die ein in einer Bilddatei enthaltenes Icon haben darf, damit es berücksichtigt wird.

Maximale Icon Höhe

Die maximale Höhe in Pixeln, die ein in einer Bilddatei enthaltenes Icon haben darf, damit es berücksichtigt wird.

Skaliere größere Icons herunter

Wenn angekreuzt, wird ein größeres Icon im Navigations-View mit der sich aus "Bevorzugte Icon Breite" und "Bevorzugte Icon Höhe" ergebenden Göße verkleinert dargestellt.
Andernfalls wird es in der originalen Größe dargestellt.

Skaliere kleinere Icons herauf

Wenn angekreuzt, wird ein kleineres Icon einer Bilddatei im Navigations-View mit der sich aus "Bevorzugte Icon Breite" und "Höhe" ergebenden Göße vergrößert dargestellt.
Andernfalls wird es in der originalen Größe dargestellt.

Bevorzugte Icon Breite

Die Breite in Pixeln, mit der das ausgetauschte Icon einer Bilddatei im Navigations-View dargestellt werden soll.

Bevorzugte Icon Höhe

Die Höhe in Pixeln, mit der das ausgetauschte Icon einer Bilddatei im Navigations-View dargestellt werden soll.
