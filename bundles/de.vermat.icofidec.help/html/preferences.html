Preferences

Here you can configure the Icon File Image Decorator and the Icon File Information Decorator according to your needs.

Enable/Disable Icon File Image Decorator

Clicking this link will take you to the "General > Appearance > Label Decorations" preference page, where you can enable or disable the Icon File Image Decorator.

Enable/Disable Icon File Information Decorator

Clicking this link will take you to the "General > Appearance > Label Decorations" preference page, where you can enable or disable the Icon File Information Decorator.

Supported file extensions

Here you can specify a list of file extensions that the decorators should accept.
The names are given without the leading "." and separated by commas. No wildcards are allowed. The names are not treated "case-sensitive".
Example: "jpg,jpeg,png".

Note: If the field is left empty, all files will be processed! This can have a negative impact on the performance of the development environment.

Included image folders

Here you can specify a list of names of folders where files will be processed. Files from other folders are ignored (even if their extensions match).
The names are separated by commas. No wildcards are allowed. The names are not treated "case-sensitive".
Example: the entry "icons,images" has the effect that files in all folders whose path contains one of the strings "icons" or "images" are processed.

Note: If the field is left empty, all files will be processed! This can have a negative impact on the performance of the development environment.

Maximum icon width

The maximum width in pixels that an icon contained in an image file may have, to be processed.

Maximum icon height

The maximum height in pixels that an icon contained in an image file may have, to be processed.

Scale down larger icons

If checked, a larger icon will be displayed in the navigation view with the size resulting from "Preferred Icon Width" and "Preferred Icon Height" scaled down.
Otherwise it will be displayed in the original size.

Scale up smaller icons

If checked, a smaller icon will be displayed in the navigation view with the size resulting from "Preferred Icon Width" and "Preferred Icon Height" scaled up.
Otherwise it will be displayed in the original size.

Preferred icon width

The width in pixels to display the replaced icon of an image file in the navigation view.

Preferred icon height

The height in pixels to display the replaced icon of an image file in the navigation view.
